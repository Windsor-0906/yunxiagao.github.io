Introduction about some HTML tags

<p>
This tag is used to define a paragraph.
<img>
This tag is used to embed images in a web page. It requires the "src" attribute to specify the image source.
<ul>
This tag is used to create an unordered list, where items are marked with bullets.
<address>
This tag is used to define contact information for the author/owner of a document.
<br>
This tag is used to insert a single line break.
